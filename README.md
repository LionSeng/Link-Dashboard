# 作品集 · 我的 AI 网页应用

一个单文件、纯前端的个人作品集页面，用来集中展示用 AI 生成的网页应用。

## 特性
- 单文件 `index.html`，双击即用，零依赖
- 作品数据存在浏览器本地（localStorage），支持在线增删改
- 按标签分组展示，支持关键词搜索
- 一键导出 / 导入 JSON 备份（方便换设备）
- 浅色 / 深色主题切换
- Stripe 风格简约设计

## 使用方法
直接用浏览器打开 `index.html` 即可。

数据保存在本地浏览器中，换设备时点击右上角「导出」备份，再到新设备「导入」。

## 部署
把 `index.html` 丢到任意静态托管（GitHub Pages / Vercel / Netlify / CloudStudio）即可上线。
